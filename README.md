# XHS 悬浮小球插件（Edge）

这是一个基于 Manifest V3 的 Edge 浏览器插件，仅在 `xiaohongshu.com` 生效。

## 功能简介

- 在小红书页面右侧显示淡蓝色悬浮小球
- 点击悬浮小球后弹出信息面板
- 解析并展示用户主页信息：
  - 昵称
  - 关注数
  - 粉丝数
  - 获赞与收藏数
  - 小红书号
  - IP 属地
  - 个性签名
- 解析并展示当前主页下的笔记列表（动态数量）：
  - 每篇笔记标题
  - 状态（如“仅自己可见”）
  - 点赞数
  - 图片链接
  - 是否自己已点赞（`#liked` 判定）
- 点击时同时在控制台输出笔记表格（`console.table`）

## 项目结构

- `manifest.json`：插件清单，声明作用域与注入脚本
- `content.js`：页面注入逻辑、DOM 解析逻辑、悬浮球与弹窗 UI

## 作用域

插件仅在以下域名运行：

- `https://*.xiaohongshu.com/*`
- `http://*.xiaohongshu.com/*`

## 安装与使用（开发模式）

1. 打开 Edge，访问 `edge://extensions`
2. 打开“开发人员模式”
3. 点击“加载解压缩的扩展程序”
4. 选择当前目录：`d:\Dev\trae_Extension`
5. 打开小红书用户主页，页面右侧出现悬浮小球
6. 点击悬浮小球查看解析结果

## 说明

- 本插件使用页面可见 DOM 结构进行提取，不依赖接口请求
- 若小红书前端结构发生较大调整，选择器可能需要同步更新
